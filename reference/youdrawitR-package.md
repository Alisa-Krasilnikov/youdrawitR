# youdrawitR: A Graphical Testing Tool Designed for Use with ggplot2

The youdrawitR package provides tools for creating interactive “You Draw
It” visualizations using *ggplot2* and *D3*. Users can convert static
plots into interactive graphics, draw trends, and generate data through
direct interaction. The package supports both guided drawing through
[`drawit()`](https://alisa-krasilnikov.github.io/youdrawitR/reference/drawit.md)
and flexible, free-form drawing through
[`sketchit()`](https://alisa-krasilnikov.github.io/youdrawitR/reference/sketchit.md).
