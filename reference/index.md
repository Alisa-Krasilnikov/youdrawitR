# Package index

## All functions

The youdrawitR package provides tools for creating interactive “You Draw
It” visualizations using ggplot2 and D3. Users can convert static plots
into interactive graphics, draw trends, and generate data through direct
interaction. The package supports both guided drawing through drawit()
and flexible, free-form drawing through sketchit().

- [`drawit()`](https://alisa-krasilnikov.github.io/youdrawitR/reference/drawit.md)
  :

  Interactive drawing on `ggplot2` visualizations using D3

- [`sketchit()`](https://alisa-krasilnikov.github.io/youdrawitR/reference/sketchit.md)
  : Free-form drawing on ggplot2 visualizations using D3

- [`ggplot_youdrawit_payload()`](https://alisa-krasilnikov.github.io/youdrawitR/reference/ggplot_youdrawit_payload.md)
  : Extract ggplot2 data for D3 rendering
